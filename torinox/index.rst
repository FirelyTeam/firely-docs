Torinox documentation
=====================

Installing
~~~~~~~~~~
Torinox is a free cross platform command line tool for small FHIR operations.
It was created by the Simplifier Developer team as a gift to the FHIR community. 

To install, go to `simplifier.net/downloads <https://simplifier.net/downloads>` and log in.

To see all commands that Torinox can run type:
::

    > fhir ?
